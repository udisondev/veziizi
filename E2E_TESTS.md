## 1. Authentication & Authorization (AuthHandler)

### 1.1 POST /api/v1/auth/login

**Описание:** Вход пользователя по email и паролю

#### Happy Path

| ID | Сценарий | Предусловия | Шаги | Ожидаемый результат |
|----|----------|-------------|------|---------------------|
| AUTH-001 | Успешный вход | Активный пользователь существует | POST с валидными email/password | 200 OK, JSON с member_id, organization_id, email, name, role. Cookie сессии установлена |
| AUTH-002 | Вход с разными ролями | Пользователи с ролями owner, administrator, member | POST для каждой роли | 200 OK, role соответствует |

#### Ошибки валидации (400)

| ID | Сценарий | Входные данные | Ожидаемый результат |
|----|----------|----------------|---------------------|
| AUTH-003 | Пустое тело запроса | `{}` | 400 Bad Request, "invalid request body" |
| AUTH-004 | Невалидный JSON | `{invalid}` | 400 Bad Request, "invalid request body" |
| AUTH-005 | Отсутствует email | `{"password": "123"}` | 400 Bad Request |
| AUTH-006 | Отсутствует password | `{"email": "test@test.com"}` | 400 Bad Request |

#### Ошибки авторизации (401)

| ID | Сценарий | Входные данные | Ожидаемый результат |
|----|----------|----------------|---------------------|
| AUTH-007 | Email не существует | `{"email": "nonexistent@test.com", "password": "any"}` | 401 Unauthorized, "invalid credentials" |
| AUTH-008 | Неверный пароль | `{"email": "existing@test.com", "password": "wrong"}` | 401 Unauthorized, "invalid credentials" |
| AUTH-009 | Регистро-зависимый email | `{"email": "TEST@test.com", ...}` | Проверить поведение (должен найти или нет?) |

#### Ошибки доступа (403)

| ID | Сценарий | Предусловия | Ожидаемый результат |
|----|----------|-------------|---------------------|
| AUTH-010 | Заблокированный аккаунт | Member со status=blocked | 403 Forbidden, "account is blocked" |
| AUTH-011 | Неактивный аккаунт | Member со status=inactive | 403 Forbidden, "account is blocked" |

#### Edge Cases

| ID | Сценарий | Описание | Ожидаемый результат |
|----|----------|----------|---------------------|
| AUTH-012 | SQL injection в email | `{"email": "'; DROP TABLE--", ...}` | 401 (безопасная обработка) |
| AUTH-013 | Очень длинный email | email > 255 символов | 400 или 401 |
| AUTH-014 | Unicode в email | `{"email": "тест@test.com", ...}` | Корректная обработка |
| AUTH-015 | Пробелы в email | `{"email": " test@test.com ", ...}` | Проверить trim |
| AUTH-016 | Пустой пароль | `{"email": "test@test.com", "password": ""}` | 401 Unauthorized |
| AUTH-017 | Огромный пароль | password > 1MB | 400 Bad Request (body limit) |

#### Фрод и безопасность

| ID | Сценарий | Описание | Ожидаемый результат |
|----|----------|----------|---------------------|
| AUTH-018 | Brute force | 10+ неудачных попыток подряд | Rate limit или блокировка |
| AUTH-019 | Логирование неудачных входов | Неверный пароль | Запись в login_history с status=failed_password |
| AUTH-020 | Логирование блокированных | Blocked аккаунт | Запись с status=failed_blocked |
| AUTH-021 | Логирование успешных входов | Успешный вход | Запись с status=success |
| AUTH-022 | Geo jump detection | Вход с другой страны | Suspicious login logged (если включен GeoIP) |

#### Конкурентность

| ID | Сценарий | Описание | Ожидаемый результат |
|----|----------|----------|---------------------|
| AUTH-023 | Параллельные входы | Одновременный вход одного пользователя | Обе сессии валидны |
| AUTH-024 | Вход во время блокировки | Блокировка аккаунта во время входа | Race condition обработан |
